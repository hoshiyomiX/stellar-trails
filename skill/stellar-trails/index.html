Stellar Trails

Stateless phase machine for LLM agents. Traceability IDs, artifact templates, source verification, and file-based memory — without shell execution or persistent hooks.

Dev server running v7.1.4 ☄️ Stateless

Phase State Machine

○
IDLE
Classify
◎
SPECIFY
Spec + SSV
◈
PLAN
IMPL-001...
◆
IMPLEMENT
Code + refs
◉
VERIFY
Tests + edges
●
DELIVER
Report
↺ Error Recovery loops back from VERIFY to SPECIFY on failure

What's Inside

🧭
Traceability IDs
IMPL-001, IMPL-002... chain through every phase — requirement → code → verification. Gaps are visible.
🎯
Adaptive Complexity
Minimal, Simple, Standard, Complex tiers. All 6 phases always run — only ceremony adjusts.
📡
Source Verification
SSV: git fetch before analysis. SADC: search existing packages + read docs before planning.
🧠
File-based Memory
Evergreen + dated files for cross-session continuity. Bounded budget, agent-curated.
🛡️
Error Decision Tree
5-step: capture → classify → identify → fix → re-verify. Covers 7 error categories.
⚡
Stateless by Design
No shell execution in Skill() invoke. Pure markdown data — agent-friendly, audit-friendly.
